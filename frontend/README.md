# Bylaw Parser Frontend

React + Vite frontend for the Bylaw Parser API.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on http://localhost:3000

## Build

```bash
npm run build
```

## Features

- Clean, minimal UI
- PDF file upload
- Loading indicator during processing
- Download buttons for extracted and translated JSON files
- Error handling
- Responsive design
